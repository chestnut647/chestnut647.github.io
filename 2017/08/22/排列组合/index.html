排列组合

发表于 2017-08-22 | 分类于 算法 |

问题描述

  • 计算下列排列组合公式，并返回组合而成的序列

  • 如[2, 3, 4]以基数2排列，则返回

解决思路

  • 这是比较典型的大问题可以化解为小问题。可以采取递归的方式来解决。
  • 首先，建立一个数组allResult用以存放结果
  • 建立一个即时运行的的递归函数
    • arr 待排列的数列
    • size 组合运算的基数
    • ==result== 是保存当前组合序列的前一部分
  • 在递归函数内部首先要对数组长度与基数进行比较
    • 当数组长度比基数小时，直接返回。
    • 当数组长度与基数相等时，说明其中一个序列获取完成，添加进allResult
  • 当数组长度比基数大时
    • 先拿到传递过来的序列的前部分，将原始序列的当前值添加至newResult
    • 当size=1时，说明序列完成，向allResult添加即可
    • size!=1，则需要进行递归，递归部分是函数最主要的部分
